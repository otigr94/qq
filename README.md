# 🎣 GTA5 RP AutoFishing Bot

**Полностью автоматический бот для рыбалки в GTA 5 RolePlay серверах**

## 🚀 Возможности

✅ **Автоматическая рыбалка** - полностью автоматизированный процесс ловли рыбы
✅ **Поддержка всех серверов** - работает на Majestic, Rage, RedLine, Diamond и других
✅ **Умное обнаружение** - компьютерное зрение (OpenCV) для определения рыбы
✅ **Реалистичные задержки** - антибан режим с случайными задержками
✅ **Полная статистика** - сохранение и анализ всех сессий рыбалки
✅ **Простой интерфейс** - удобное меню с настройками
✅ **Многоязычность** - поддержка русского языка

## 📋 Требования

- **Python 3.7+**
- **Windows 10/11** (для других ОС нужны адаптации)
- **Экран разрешением 1920x1080** (или измени в config.py)
- **GTA 5 с мультиплеером РП мод**

## ⚙️ Конфигурация

### config.py

**Основные параметры:**
```python
MODE = 'auto'  # Режимы: auto, manual, smart

CLICK_DELAY = 100  # Задержка между кликами (мс)
CATCH_DELAY = 2.0  # Задержка после поимки (сек)
READY_DELAY = 1.0  # Задержка готовности (сек)

FISH_DETECTION_THRESHOLD = 0.6  # Порог обнаружения (0-1)
```

**Поддерживаемые серверы:**
```python
SUPPORTED_SERVERS = [
    'Majestic',
    'Rage',
    'RedLine',
    'Diamond',
    'Elite',
    'Fantasy'
]
```

**Горячие клавиши:**
```python
STOP_KEY = 'q'    # Остановка
PAUSE_KEY = 'p'   # Пауза
RESUME_KEY = 'r'  # Возобновление
```

## 🐛 Решение проблем

### Бот не обнаруживает рыбу
1. Проверь что ты в режиме рыбалки в игре
2. Увеличь `FISH_DETECTION_THRESHOLD` в config.py
3. Включи `SHOW_DETECTION_WINDOW = True` для отладки

### Неправильные задержки
1. Отредактируй `CLICK_DELAY` в config.py
2. Используй конфигурацию конкретного сервера
3. Проверь `get_server_config()` для твоего сервера

### Бот работает медленно
1. Уменьши `CLICK_DELAY`
2. Отключи отладку: `DEBUG_MODE = False`
3. Закрой лишние приложения
